# Frame & Function — modern, minimalist portfolio site blueprint (save as .txt)

This blueprint is a practical, implementation-ready spec for rebuilding Frame & Function: simple, modern, visually beautiful, and focused on clean presentation of digital design services. You can paste this into a .txt file in VS Code and build directly from it.

---

## Goals and vibe

- **Purpose:** Showcase your design work, services, and process with a clean, modern, fast site.
- **Tone:** Minimalist, editorial, confident. High-contrast typography, generous whitespace, crisp micro-interactions.
- **References:** Contemporary portfolio patterns (Wix inspiration for showcase), trending design aesthetics: neutral palettes + a single accent, subtle grain/noise, smooth scroll, understated motion.

---

## Information architecture

- **Top-level navigation:**
  - **Home** — hero, value prop, signature work highlights, social proof.
  - **Work** — case studies and project cards with filters.
  - **Services** — clear, modular offerings with transparent outcomes.
  - **About** — philosophy, approach, bio, selective clients.
  - **Contact** — simple inquiry form with project intake.
  - Optional: **Notes** (short essays/process posts) for authority and SEO.

- **Footer:**
  - Brand mark, email, social links, location (Charlotte, NC), copyright.
  - Quick links: Work, Services, About, Contact.

---

## Visual style and UI system

- **Color palette (tokens):**
  - **Base:** #0A0A0A (ink), #111315 (deep), #F6F7F8 (paper), #DDE1E6 (subtle grey)
  - **Accent:** #6EE7B7 (mint) or #A78BFA (lilac) — choose one
  - **State:** success #16A34A, warning #F59E0B, error #EF4444

- **Typography:**
  - **Display:** Inter Tight / Satoshi / ABC Ginto Nord (any modern grotesk)
  - **Body:** Inter / Geist / General Sans
  - **Scale:** clamp-based fluid types
    - H1: clamp(32px, 6vw, 64px)
    - H2: clamp(24px, 4vw, 40px)
    - Body: clamp(16px, 2vw, 18px)
  - **Tracking:** Slightly tightened for headings (-1% to -2%), normal for body.

- **Layout:**
  - Grid: 12-column, 72–96px max container padding, 1200–1440px content width.
  - Whitespace: generous sections, 80–120px vertical rhythm.
  - Cards: soft shadow or border with .5px stroke (#E5E7EB), subtle radius (6–10px).

- **Motion:**
  - Hover: 120–180ms ease-out scale/translate, subtle shadow.
  - Entrance: 280–360ms fade+translate, staggered by 60ms.
  - Parallax: extremely subtle (2–6px) on hero imagery; reduce motion for prefers-reduced-motion.

- **Textures:**
  - Optional fine grain/noise layer at 2–4% opacity on hero/backgrounds.

## Content blueprint

- **Home hero:**
  - Headline: “Modular digital design. Functional by default.”
  - Sub: “I design and implement lean, modern web experiences—crafted for clarity, speed, and control.”
  - CTAs: View work (primary), Contact (secondary)
  - Visual: marquee strip of project thumbnails or a single editorial image.

- **Work highlights:**
  - 6–9 featured projects with quick stats (role, deliverables, stack).
  - Filter chips: Website, Identity, UI/UX, Systems.
  - Quick view modal with 3–5 images, summary, link to full case.

- **Services:**
  - Cards (3–5): Web build, Identity system, UX frameworks, Design ops.
  - Each card: outcome, inclusions, timeline range, starting price (optional), CTA.

- **About:**
  - Short portrait paragraph, working principles (clarity, modularity, ownership).
  - Select clients or sectors, tool stack, location (Charlotte, NC).

- **Contact:**
  - Inputs: name, email, company, budget range, timeline, message.
  - Optional: checkbox “Attach brief” (file upload).
  - Success state: Thank-you + next steps expectations (48h reply, initial call).

## Accessibility, SEO, and performance

- **Accessibility:**
  - Color contrast AA minimum; skip-to-content link; semantic landmarks; labeled form inputs; focus states visible.
  - Alt text on all visuals; reduced motion respect; keyboard navigation tested.

- **SEO:**
  - Clean titles/meta; Open Graph + Twitter cards; sitemap.xml and robots.txt.
  - Structured data: Organization + WebSite + BreadcrumbList; Article schema for case studies.

- **Performance:**
  - Image optimization (AVIF/WebP + responsive srcset).
  - CSS-first interactions; no heavy libraries; defer non-critical scripts.
  - Lighthouse targets: 90+ across metrics.

## Animations and interactions

- **Card hover:** translateY(–2px), shadow increase, image scale 1.01.
- **Section reveal:** IntersectionObserver triggers fade+translate up 12px.
- **Filter chips:** active state with accent underline; animate layout with CSS transitions.
- **Modal:** body scroll lock; escape-to-close; focus trap; aria-modal.

---

## Build, deploy, and CMS options

- **Static-first:** Vite + React, deploy on Netlify/Vercel.
- **CMS options:** Markdown files (fast), or a lightweight headless CMS (Sanity/Contentful) if you want client-friendly updates.
- **Performance checklist:** preconnect fonts, compress images, lazy-load below the fold, tree-shake, minify, use modern formats.
